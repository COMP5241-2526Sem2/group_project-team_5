# Quiz 模块设计方案（参考 xiti.ai · 初高中版）

**参考产品**：[习题AI](https://xiti.ai/)

**目标用户**：初高中学生（12-18岁）及教师

**核心特点**：

- 覆盖中考、高考备战场景
- 支持语数英物化生政史地等全学科
- 题型丰富（选择题、填空题、阅读理解、完形填空、作文等）
- 错题本与薄弱点分析
- 学习进度追踪

---

## 1. 产品定位

### 1.1 核心功能矩阵（对齐 xiti.ai 子页面）


| xiti.ai 页面                                  | 功能      | 说明                 | 优先级 |
| ------------------------------------------- | ------- | ------------------ | --- |
| [AI 出题](https://xiti.ai/exercise/generator) | AI 出题   | 多源输入 → AI 生成练习题    | P0  |
| [AI 出卷](https://xiti.ai/exam/generator)     | AI 出卷   | 设置试卷类型 → 生成完整试卷    | P1  |
| [题库](https://xiti.ai/exercise/library)      | 题库管理    | 题目筛选、分类、复用         | P0  |
| [试卷库](https://xiti.ai/exam/library)         | 试卷管理    | 试卷列表、模板管理          | P1  |
| [练习册](https://xiti.ai/workbook/generator)   | AI 出练习册 | 按章节/专题编排练习册        | P2  |
| -                                           | 错题本     | 自动收集错题、针对性复习       | P0  |
| -                                           | 作答与批改   | AI 自动批改客观题，教师批改主观题 | P0  |
| -                                           | 结果分析    | 正确率、薄弱点、改进建议       | P1  |


### 1.2 用户角色

- **教师**：AI 出题、题库管理、AI 出卷、练习册编排、批改主观题/作文
- **学生**：作答 Quiz、查看结果、错题本复习

### 1.3 学科与年级体系

```
学段：
├── 初中（7-9年级）
│   ├── 初一 / 七年级
│   ├── 初二 / 八年级

│
└── 高中（10-12年级）
    ├── 高一 / 十年级
    ├── 高二 / 十一年级

├── 语文、数学、英语

├── 思想政治、历史、地理（文科）
└── 音乐、美术、体育、信息技术（素质科目）
```

## 2. 数据模型设计

### 2.1 题目（Question）

```typescript
interface Question {
  id: number;
  quiz_id: number | null;
  
  // 题干
  prompt: string;
  image_url: string | null;
  
  // 题型
  q_type: QuestionType;
  
  // 选项
  options: QuestionOption[];
  
  // 评分
  difficulty: 'easy' | 'medium' | 'hard';
  score: number;
  explanation: string | null;
  
  // 元数据
  subject: string;
  grade: number;
  chapter: string;
  knowledge_points: string[];
  tags: string[];
  source_refs: SourceRef[];
  generated_by: 'ai' | 'manual';
  
  // 状态
  status: 'draft' | 'published';
  
  created_at: string;
  updated_at: string;
}

type QuestionType = 
  | 'MCQ_SINGLE'           // 单选题
  | 'MCQ_MULTIPLE'         // 多选题
  | 'TRUE_FALSE'          // 判断题
  | 'FILL_BLANK'          // 填空题
  | 'CLOZE'               // 完形填空
  | 'READING_COMPREHENSION' // 阅读理解
  | 'SHORT_ANSWER'        // 简答题
  | 'ESSAY'               // 作文
  | 'LISTENING'           // 听力题
  | 'CALCULATION';        // 计算题

interface QuestionOption {
  option_key: string;
  option_text: string;
  is_correct: boolean;
}

interface SourceRef {
  type: 'text' | 'document' | 'textbook' | 'exam_paper';
  content?: string;
  document_id?: number;
  page_number?: number;
  textbook_info?: {
    name: string;
    publisher: string;
    chapter: string;
  };
}
```

  sections: PaperSection[];
  question_ids: number[];

  score_each: number;

}

interface Submission {
