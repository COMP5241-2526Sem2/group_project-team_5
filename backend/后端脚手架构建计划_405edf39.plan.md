# 后端脚手架构建计划

## 技术栈

- **运行时**: Python 3.11+
- **框架**: FastAPI + Uvicorn
- **ORM**: SQLAlchemy 2.0 (async)
- **数据库**: MySQL (dev: SQLite via aiosqlite)
- **认证**: JWT (python-jose) + bcrypt (passlib)
- **API 文档**: Swagger (FastAPI 内置, OpenAPI 3.0)
- **验证/序列化**: Pydantic v2
- **数据库迁移**: Alembic
- **测试框架**: pytest + pytest-asyncio
- **测试覆盖**: pytest-cov
- **测试数据**: Faker (faker)
- **包管理**: uv (优先) 或 pip

## 数据模型（SQLAlchemy ORM — 对应前端推断的实体）

### User

```python
User: id, account_id, hashed_password, name, account_type(ENUM: STUDENT/TEACHER/ADMIN),
      phone, id_card, registered_at, last_login, accessibility?
Student: student_id, department, major, homeroom  # extends User
Teacher: employee_id, department  # extends User
```

### Course

```python
Course: id, name, subject, grades(JSON), period, room, weekdays,
        max_students, status, teacher_id
Enrollment: id, student_id, course_id, enrolled_at
```

### Quiz

```python
Quiz: id, title, course_id, due_at, duration_min, question_count, status
Question: id, quiz_id, order, type(ENUM: MCQ_SINGLE/SHORT_ANSWER),
          prompt, image?
QuestionOption: id, question_id, key, text
QuizAttempt: id, quiz_id, student_id, score, submitted_at, answers(JSON)
Answer: id, attempt_id, question_id, mcq_choice?, sa_text?, is_correct?
```

### Assessment (试卷)

```python
Paper: id, title, subject, grade, semester, type, total_score,
       duration_min, status, teacher_id
PaperSection: id, paper_id, title, order
QuestionSet: id, section_id, type, difficulty, chapter, questions(JSON)
```

### Lesson (课件)

```python
LessonDeck: id, title, subject, grade, status(ENUM: DRAFT/PUBLISHED),
            teacher_id, created_at, updated_at
Slide: id, deck_id, title, order
SlideBlock: id, slide_id, type(ENUM: TEXT/INTERACTIVE/EXERCISE), content, order
```

### Lab

```python
LabRegistry: id, registry_key, title, subject, type, renderer_profile,
             initial_state(JSON), reducer_spec(JSON), metadata(JSON),
             status, teacher_id?
```

## 关键设计决策

### API 响应格式（统一）

FastAPI 自动返回 Pydantic 模型作为 JSON，格式如下：

```json
{ "success": true, "data": {...}, "message": "..." }
{ "success": false, "error": { "code": "...", "message": "..." } }
```

所有 API 路由通过 Pydantic `ResponseModel` 统一响应结构。

### 角色权限矩阵


| 端点                            | Student | Teacher | Admin |
| ----------------------------- | ------- | ------- | ----- |
| `POST /api/v1/auth/*`         | ✅       | ✅       | ✅     |
| `GET /api/v1/admin/users/*`   | ❌       | ❌       | ✅     |
| `POST /api/v1/teacher/decks`  | ❌       | ✅       | ❌     |
| `GET /api/v1/student/quizzes` | ✅       | ❌       | ❌     |


### API Base URL 配置

后端在 `.env` 中配置 `API_BASE_URL=http://localhost:8000`，前端在 `vite.config.ts` 中配置代理将 `/api` 请求转发到后端。

### AI API 配置（OhMyGPT）

所有 AI 相关功能（课件 AI 辅助、试卷生成、Lab 生成）统一调用 OhMyGPT API，密钥严格从环境变量读取。

```toml
# .env.example — 所有密钥类变量放在此处，不提交到仓库
DATABASE_URL=sqlite+aiosqlite:///./data/dev.db
# 生产环境改为: mysql+aiomysql://user:pass@host:3306/dbname

JWT_SECRET_KEY=your-super-secret-jwt-key-change-in-production
JWT_ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=1440  # 24h

# OhMyGPT API（AI 功能）
OHMYGPT_API_KEY=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
OHMYGPT_BASE_URL=https://api.ohmygpt.com/v1
```

**安全规则**：

- `.env` 文件已加入 `.gitignore`，禁止提交
- `.env.example` 只含变量名不含实际值，提交到仓库作为模板
- 密钥在任何日志、错误响应、print 语句中均不得出现
- `app/config.py` 通过 `pydantic-settings` 读取，所有配置以 `settings.xxx` 形式使用
- OhMyGPT API 调用封装在 `app/core/ai_client.py`，业务层不得直接传入密钥

```python
# app/core/ai_client.py — AI 调用封装示例
from openai import AsyncOpenAI
from app.config import settings

ai_client = AsyncOpenAI(
    api_key=settings.ohmygpt_api_key,
    base_url=settings.ohmygpt_base_url,
)

async def generate_text(prompt: str, model: str = "gpt-4o-mini") -> str:
    response = await ai_client.chat.completions.create(
        model=model,
        messages=[{"role": "user", "content": prompt}],
    )
    return response.choices[0].message.content
```
